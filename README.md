# Flare Demo

**You Are a Developer.** A presentation for healthcare founders on how AI tools are making everyone a builder.

🔗 **[View the presentation](https://e-straight.github.io/flare-demo/)**

## About

This is a Reveal.js presentation built for a Flare Capital event. The core message: anyone can be a developer — the tools have caught up to you. Domain experts (like healthcare founders) are uniquely positioned to build the most useful software because they already understand the problem space.

## Tools Featured

- [GitHub Copilot CLI](https://github.com/features/copilot/cli/) — Build software from your terminal using natural language
- [GitHub Spark](https://github.com/features/spark) — Go from idea to working app with no code, no setup
